# Uknuta Magia

**A lightweight experimental e-commerce project built for occasional book sales without a formal company setup.**

Uknuta Magia is a small personal web shop created to help an author sell printed copies of his book as occasional sales.  
The project includes a simple storefront and an admin panel connected to Supabase for managing orders and products.

Built as a practical portfolio project and real-world experiment.
